Политика конфиденциальности

1. Настоящая Политика определяет порядок обработки персональных данных в соответствии с ФЗ-152 «О персональных данных».

2. Сайт собирает email, имя, телефон, адрес доставки, историю заказов. Данные используются только для обработки заказов.

3. Передача данных третьим лицам возможна только для исполнения договора (доставка, оплата) или по запросу органов.

4. Пользователь вправе запросить удаление данных, направив запрос на support@tapkidrop.ru.
